使用指南

第一步：电源连接

  1. 确认电源适配器输出为 DC 5V-12V
  2. 将电源适配器插头插入设备电源接口
  3. 将电源适配器连接到交流电源插座
  4. 此时设备处于待机状态，显示屏不亮

第二步：开启系统

  1. 找到设备上的主按键
  2. 短按主按键（按下后立即松开）
  3. 显示屏亮起，显示系统初始化信息
  4. 传感器开始采集数据，设备进入工作状态

第三步：查看显示屏

显示屏会实时显示以下信息：

  • 第一行：系统状态（电源图标）+ 当前模式（AUTO/MANUAL）
  • 第二行：LED灯状态（ON/OFF）+ 风扇状态（ON/OFF）
  • 第三行：光照强度：XXX lux
  • 第四行：温度：XX.X°C
  • 第五行：网络状态 + IP地址（连接成功后显示）

第四步：模式切换

切换到手动模式

  1. 找到设备上的模式切换键
  2. 长按模式切换键（保持按下状态超过2秒）
  3. 显示屏上模式指示变为"MANUAL"
  4. 系统进入手动控制模式

切换回自动模式

  1. 再次长按模式切换键（超过2秒）
  2. 显示屏上模式指示变为"AUTO"
  3. 系统进入自动控制模式

第五步：手动控制（手动模式下）

在手动模式下，短按模式切换键可循环切换四种设备状态：

  1. 状态1：灯灭，风扇停 → 短按 → 状态2
  2. 状态2：灯亮，风扇停 → 短按 → 状态3
  3. 状态3：灯灭，风扇转 → 短按 → 状态4
  4. 状态4：灯亮，风扇转 → 短按 → 状态1

注意：每次按键间隔应大于1秒，避免误操作。

第六步：网络连接

  1. 确保设备已开启并处于正常工作状态
  2. 设备会自动搜索附近的WiFi网络
  3. 在设备配置界面选择目标WiFi网络（首次使用需配置）
  4. 输入WiFi密码后等待连接
  5. 连接成功后，显示屏会显示设备IP地址

第七步：关闭系统

  1. 找到设备上的主按键
  2. 短按主按键（按下后立即松开）
  3. 显示屏熄灭，系统进入待机状态
  4. 如需完全断电，可拔掉电源适配器

安全注意事项

  • 电源检查使用前检查电源适配器输出电压是否符合要求
  • 环境要求避免将设备放置在高温、潮湿或有腐蚀性气体的环境中
  • 安全操作不要用湿手操作设备或插拔电源
  • 风扇安全设备工作时风扇会转动，请勿将手指或异物伸入风扇
  • 断电保存长时间不使用时，请断开电源
  • 清洁保养清洁设备时，请先断开电源

常见问题排查

设备无法开启

  • 检查电源适配器是否正确连接
  • 检查电源插座是否有电
  • 检查电源适配器输出是否正常

显示屏不显示

  • 确认系统已开启（主按键已按下）
  • 检查显示屏连接线是否松动
  • 尝试重启设备

传感器数据异常

  • 检查传感器连接线是否正确
  • 确认传感器供电正常
  • 尝试重新校准传感器

网络无法连接

  • 检查WiFi密码是否正确
  • 确认WiFi信号强度足够
  • 尝试靠近路由器
